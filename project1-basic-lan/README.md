# Project 1 – Basic LAN Network

## Objective
Build a simple Local Area Network (LAN) using a Cisco switch and two PCs.

## Topology
PC1 --- Switch --- PC2

2 PCs
1 Cisco 2960 Switch

## IP Addressing

PC1
IP: 192.168.1.10
Subnet Mask: 255.255.255.0

PC2
IP: 192.168.1.11
Subnet Mask: 255.255.255.0
