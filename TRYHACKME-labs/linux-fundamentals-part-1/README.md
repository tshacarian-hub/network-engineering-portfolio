# TryHackMe — Linux Fundamentals Part 1

## Objective
Build core Linux command-line skills by learning how to navigate the file system, manage files and directories, and understand basic shell usage.

---

## Tools Used
- pwd
- ls
- cd
- cat
- touch
- mkdir
- cp
- mv
- rm
- grep
- TryHackMe virtual machine

---

## What I Did
- Navigated the Linux file system using common shell commands
- Listed files and directories with ls
- Changed directories with cd
- Viewed file contents with cat
- Created files and folders with touch and mkdir
- Copied, moved, and removed files using cp, mv, and rm
- Completed hands-on Linux tasks in the TryHackMe lab environment

---

## Key Concepts Learned
- Linux uses a hierarchical file system
- The terminal is used to interact with the system through commands
- pwd shows the current working directory
- ls lists files and folders in a directory
- cd changes the current directory
- cat displays file contents
- touch creates empty files
- mkdir creates directories
- cp copies files and folders
- mv moves or renames files and folders
- rm deletes files and folders
