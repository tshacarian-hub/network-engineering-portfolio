# TryHackMe — Introductory Networking

## Objective
Learn the core concepts of networking, including IP addressing, DNS resolution, and basic network troubleshooting techniques.

---

## Tools Used
- whois  
- dig  
- ping  
- traceroute  
- TryHackMe virtual machine  

---

## What I Did
- Identified domain ownership and registration details using whois  
- Queried DNS records using dig to understand how domain names resolve  
- Tested network connectivity using ping  
- Traced the path packets take across networks using traceroute  
- Completed hands-on tasks in the TryHackMe lab environment  

---

## Key Concepts Learned
- IP addresses uniquely identify devices on a network  
- DNS translates domain names into IP addresses  
- whois provides domain registration and ownership details  
- dig is used to query DNS servers and view DNS records  
- ping uses ICMP to test connectivity between devices  
- traceroute shows each hop a packet takes to reach its destination

## Real-World Relevance
These networking fundamentals are used in IT support and network administration to:

- Troubleshoot connectivity issues using ping  
- Diagnose DNS problems using dig  
- Analyze network paths and latency using traceroute  
- Investigate domain ownership and legitimacy using whois
